Skills Required

Since Android and iOS operating systems are essential creating the app, basic knowledge about their programming languages Java and Swift is required. Without studying one of these two languages, it is impossible to forge any apps or website. Aside from that, I need to learn about designing tools so that the game has a livelier and more enticing graphics. For the pet’s behavior to be more intelligent, it requires some codes. For this I will use AI. More specifically, I will create an AI model and then write codes in Python to control it. Hence, some skills in Python is fundamental as well. Next, in order to utilize the model to create a hybrid one, I need to know about Restful API, as well as JavaScript to use them in Restful API. Consequently, another programming language that this app requires me to look into is JavaScript.

Outcomes

Mental illness is a condition that can impact deeply into a person’s personal and professional life. Most of the times this condition is kept secretly as people are fear of judgements and criticism. Thus if this project is to be successful, not only it will impact myself as a creator but also to everyone who is suffering mental illness. They need to know that they are not alone because help exists, and that is one of the purpose of this app. It can become a close friend, a guider, an advisor to minimize the symptoms and help them have a normal life like everybody else
